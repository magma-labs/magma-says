# magma-says
A magam implementation for classic cowsay

## Usage
Add in your `Cargo.toml`:

```
[dependencies]
magma-says = "0.1"
```

Then in your main file or where you plan to use:

```rust
use magma_says;

fn main() {
    let out = "Hello Rustaceans!";

    magma_says::say(out);
}
```
